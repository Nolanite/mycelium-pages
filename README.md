# Mycelium OS GitHub Pages Starter

## Features

- Markdown-based lessons
- Callouts
- Progressive reveal with `<details>` tags
- Interactive node demo with D3.js
- Alpine.js for hint toggles
- Single-page structure, easy to extend
- Mycelium-style modular layout

## Usage

1. Clone this repo
2. Add new lessons in `lessons/`
3. Link them in `index.html`
4. Push to GitHub and enable GitHub Pages
5. Your site is live at `https://<username>.github.io/<repo>/`
